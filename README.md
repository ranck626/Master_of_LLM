## Master of LLM Knowledge

### 1. 项目背景

​	在技术岗位面试中，HR或面试官常会提出一些“标准化”的技术问题，逐渐形成了所谓的“八股”模式。然而，通用大模型在回答这些问题时，往往扩展过多、抓不住重点，难以满足面试对简明精炼的要求。为此，我们通过RAG、SFT等技术，专门训练一个聚焦LLM领域知识的“八股”大模型——“LLM八股仙人”，帮助面试者高效备考、精准练习面试回答。
